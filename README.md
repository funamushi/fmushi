## 開発環境

```bash
$ npm install -g brunch
$ npm install -g bower
```

```bash
$ git clone git@github.com:funamushi/fmushi.git
$ cd fmushi
$ npm install
$ brunch w -s
```

## とすて

```bash
$ npm test
```

## デプロイ

```bash
$ bundle exec cap staging deploy
```
